# Timer

大学の演習課題として、M5Stack Core2 for AWSを用いた、時計、タイマー、ストップウォッチ、テトリスなどの機能を持つ、多機能デバイスを制作しました。
題材として、[IKEAのKLOCKIS](https://www.ikea.com/jp/ja/p/klockis-clock-thermometer-alarm-timer-white-50277005/)を参考にしました。
